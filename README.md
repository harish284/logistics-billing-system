# Logistics Billing System

A comprehensive, full-stack logistics and freight management SaaS application. 

This system is designed to handle custom rate cards, dynamic shipment calculations, customer management, and professional PDF invoice generation. 

## Key Features
* **Advanced Dashboard:** Real-time analytics, dynamic month-over-month growth calculations, and active client tracking.
* **Customer & Rate Cards:** Complete management of clients and customizable origin-to-destination pricing.
* **Shipment Processing:** Automated cost calculation based on weight and active rate cards.
* **Professional Invoicing:** Fully dynamic invoice builder with line-item calculations, tax/discount support, and automated high-quality A4 PDF generation using Puppeteer.

## Tech Stack
* **Frontend:** React, Vite, Tailwind CSS, Recharts
* **Backend:** Node.js, Express, Prisma ORM, Neon PostgreSQL, Puppeteer
